Attribute what="what"

what is a list of elements, which define the extent of the answers to a <show_...> command. The elements are separated with commas. This list can be empty.

Examples

<show_state what="orders"/>

<show_state what="task_queue,orders=10"/>

Elements


all

A combination of independent, non-XML elements.

task_queue

Returns the <task_queue>.

orders

Returns the orders (<order>)

orders=limit

Returns the first limit orders.

description

Returns the <description>.

log

Returns the <log>.
