Attribut what="what"

what ist eine Liste von Elementen, die den Umfang der Antwort auf ein <show_...>-Kommando bestimmen. Die Elemente werden durch Komma getrennt. Die Liste kann leer sein.

Beispiele

<show_state what="orders"/>

<show_state what="task_queue,orders=10"/>

Elemente


all

Abhängig vom XML-Elemente eine Kombination anderer Elemente.

task_queue

Liefert die <task_queue>.

orders

Liefert die Aufträge (<order>)

orders=limit

Liefert die Aufträge, aber nicht mehr als limit Stück.

description

Liefert <description>.

log

Liefert <log>.
